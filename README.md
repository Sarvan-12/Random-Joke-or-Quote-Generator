# Random Joke and Motivation App

A simple Flask web app that shows either:
- A **Random Joke** (for fun)
- A **Motivational Quote** (for inspiration)

---

## Features

- Get a random joke 🃏
- Get a motivational quote ✨
- Simple and clean interface

---

## APIs Used

- [Official Joke API](https://official-joke-api.appspot.com/)
- [Zen Quotes API](https://zenquotes.io/)

---

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Sarvan-12/Random-Joke-or-Quote-Generator.git
   cd Random-Joke-or-Quote-Generator
   ```

2. Create and activate a virtual environment:
   ```sh
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Mac/Linux
   source venv/bin/activate
   ```

3. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

## Running the App

```sh
python app.py
```
The app will be available at [http://127.0.0.1:5000](http://127.0.0.1:5000).
